./echo.elf
  -> execute l'interp en mode REPL

./echo.elf -
./echo.elf srcfile
  -> execute l'interp

./echo.elf\ -i -
./echo.elf\ -i srcfile
  -> execute l'interp en mode interactive

---

./echo.elf --
  -> execute l'embed (echo.ml)

./echo.elf -- <arg(s)>
  -> execute l'embed

./echo.elf\ -i -- <arg(s)>
  -> execute l'interp en mode REPL (présence du -i)
     (utile pour passer des arguments en mode REPL)
---

./interp.elf -- <arg(s)>
  -> fallback sur le mode REPL
     (requiert le ELF original, ne fonctionne pas sur un ELF embeddé)
